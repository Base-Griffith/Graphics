# Usage

refactor.py [-h] [-c COLOR] [-p PATTERN] [-f FILE]

**Shade arbitrary closed regions**

optional arguments:
  -h, --help            show this help message and exit   
  
  -c COLOR, --color COLOR
                        Choose a color   
                        
  -p PATTERN, --pattern PATTERN
                        Choose a shading pattern   
                        
  -f FILE, --file FILE  Choose an input file
